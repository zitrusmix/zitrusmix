# Zitrusmix

## Constructor

```TypeScript
new Zitrusmix();
```

## Methods

### add
```TypeScript
Zitrusmix.prototype.add(uri: ElementURI, content?: Content): ContentElement
```

Add `content` to the `Zitrusmix`. The `content` can be any serializable object.
Returns the `ContentElement`.
